Hashing and encoding functions for turbopack.

An example use of this module is hashing a file’s content for cache invalidation, and encoding the hash to an hexadecimal string for use in a file name.

  • DeterministicHasher
    DeterministicHasher is a custom trait that signals the implementor can safely hash in a replicatable way across platforms and process runs. Note that the default Hasher trait used by Rust allows for non-deterministic hashing, so it is not suitable for our purposes.
